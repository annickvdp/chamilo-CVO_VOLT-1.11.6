ActivePresenter Slideshow

ActivePresenter is een programma dat je kan gebruiken om schermopnames te maken, dit kan helpen om de theorie uit te leggen. Je kan bijvoorbeeld een scheropname maken van een Powerpoint en deze exporteren als video (zie vorige pagina).

ActivePresenter kan voor zoveel meer gebruikt worden dan alleen schermopnames.  Je kan er ook slides bewerken, interactieve elementen toevoegen, etc.

In het voorbeeld hieronder werd een ActivePresenter geëxporteerd als "HTML5 simulation".  Je zou van ditzelfde project ook gewoon een video kunnen maken (zie voorbeeld en handleiding bij rubriek "video").
