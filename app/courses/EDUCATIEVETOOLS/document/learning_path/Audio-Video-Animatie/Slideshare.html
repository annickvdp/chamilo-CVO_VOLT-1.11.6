Slideshare

Powerpoint slideshows insluiten op een pagina kan via "Slideshare".  Je kan op slideshare.com de powerpoint opladen en daar de code kopiëren die je nodig hebt om de slideshow  in te sluiten op de pagina.

Elke slide wordt wel als statische slide getoond, alle animaties verdwijnen dus. Daarom raden we deze methode af, kies dan liever voor "Powerpoint insluiten via OneDrive".
