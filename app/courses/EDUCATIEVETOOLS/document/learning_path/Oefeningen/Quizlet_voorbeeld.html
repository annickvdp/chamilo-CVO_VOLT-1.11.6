Quizlet voorbeeld

Ofwel embed je in Chamilo (zie hieronder), dat kan alleen als de flashcards publiek zijn. (maar dat is geen probleem denk ik).  

Ofwel kan je linken naar quizlet:  http://quizlet.com/_poyfg.  In dit geval kan je de kaarten beschikbaar stellen met paswoord of aan een bepaalde klas en dan kan je als je een betalend account hebt de scores van je cursisten zien.

Deze set werd met de volgende html code op deze pagina ingesloten. Je kan de hoogte en de breedte in de code aanpassen. (hieronder is de breedte 800 en de hoogte 460)

          <iframe src="https://quizlet.com/43154764/flashcards/embedv2" height="460" width="800" style="border:0;"></iframe>

Met de link "View this study set" linksonder, kan je de woorden ook in een lijst en in een volledig scherm bekijken.
Met het keuzemenu "study mode" rechtsonder, kan je een oefenmethode kiezen om de zinnen in te oefenen.  
Je kan ook de mobiele app Quizlet downloaden en de sets op een tablet of smartphone bekijken.
