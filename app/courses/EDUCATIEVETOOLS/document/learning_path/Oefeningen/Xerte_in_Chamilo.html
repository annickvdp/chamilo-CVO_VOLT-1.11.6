Xerte in Chamilo

 
Eens je een Xerte-object gemaakt heb op Toll-net,
kan je dit leerobject integreren in Chamilo.
 
Dit kan op twee verschillende manieren. Lees meer over de methodes hieronder en klik op de handleiding "Xerte in Chamilo" om te lezen hoe je concreet te werk gaat.
 
 
METHODE 1: ALS SCORM - Klik hier voor een voorbeeld
Je kan een Xerte leerpad bewaren als SCORM object. SCORM is een afkorting van Sharable Content Object Reference Model, vrij vertaald dus een model voor het uitwisselen van brokjes ‘leerinhoud’. Dit Xerte (SCORM) leerpad kan je dan importeren. 
Voordeel van deze werkwijze is dat je de score (alleen totaal score) van het leerpad kan bekijken (maar geen antwoorden). Nadeel is dat elk Xerte leerpad een apart leerpad moet zijn, dus in een cursus waarbij leerpaden gebruikt worden is dit niet handig, want je kan een xerte leerpad op deze manier niet IN een bestaand Chamilo leerpad gebruiken. 
Noot: eens een Xerte leerpad werd geïmporteerd in Chamilo, kan je wel andere elementen toevoegen in dit leerpad.
 
METHODE 2: MET EEN IFRAME - voorbeeld in dit leerpad onderaan
Of je kan een Xerte leerpad van op de Toll-net website in een Chamilo leerpad insluiten met een iframe. De cursist raadpleegt dan eigenlijk rechtstreeks het Xerte leerpad op de server van Toll-net. Een voordeel is dat als je dus iets verandert in het leerpad bij Toll-net, die veranderingen meteen ook zichtbaar zijn in elke Chamilo cursus. Bovendien kan je nu ook een Xerte leerpad gebruiken in een bestaand Chamilo leerpad. Let wel!: Je ziet geen scores, alleen een indicatie van de tijd die een cursist doorbracht op deze pagina (dus voor het volledige Xerte-object).
