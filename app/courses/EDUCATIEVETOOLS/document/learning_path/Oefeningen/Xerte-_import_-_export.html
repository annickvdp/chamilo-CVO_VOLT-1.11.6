Xerte: import - export

 
In de handleiding van Toll-net staat beschreven hoe je online via de installatie van Xerte op de Toll-net-server Xerte-objecten met elkaar kan delen of aan elkaar kan doorgeven.
 
Maar wat met leerobjecten op Klascement bijvoorbeeld?
 
EXPORT
 
Werkwijze om een Xerte-object te exporteren om bijvoorbeeld op Klascement op te laden:
  • selecteer het Xerte-object en klik op "Eigenschappen"
  • klik op de tab "Export"
  • klik op de knop "Maak zip bestand"

 

IMPORT

Werkwijze om een Xerte zip-bestand te importeren om bijvoorbeeld Xerte-objecten van Toll-net te importeren.

  • selecteer de map "werkruimte"

     
  • klik op "Eigenschappen"
  • klik op de tab "Import"
  • klik op "Bestand kiezen" en blader naar het zip-bestand dat je wilt importeren
  • Typ een naam voor het Xerte-object
  • klik "Upload"
